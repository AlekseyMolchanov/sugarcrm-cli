# Sugar CRM comman line interface

[![CircleCI](https://circleci.com/gh/AlekseyMolchanov/sugarcrm-cli.svg?style=svg)](https://circleci.com/gh/AlekseyMolchanov/sugarcrm-cli)

# Usage
  
  ./sugar_cli {module} {action} (--params ...... +)
  
  Modules:
  
    * Account
    * Contact    
  
  Actions:
    
    * show
    * get
    * create
    * update
    * delete
    * cascade_delete
